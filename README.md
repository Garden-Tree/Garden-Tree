# Hi there, I'm GardenTree 👋

長崎大学大学院で情報工学を専攻している修士1年生（M1）です。

**「ハードウェアからインフラの裏側まで。技術と人を繋ぎ、『生きたシステム』を実装する。」**
をモットーに、特定の領域に縛られず、目の前の課題を解決するために最適な技術を選び、形にすることを大切にしています。

👉 **これまでの開発プロジェクトや経歴、受賞歴については [ポートフォリオサイト](https://gardentree-portfolio.vercel.app/) をご覧ください！**
